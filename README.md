Este projeto foi desenvolvido como parte dos meus estudos como Desenvolvedor Mobile Flutter. O objetivo desse projeto foi criar um aplicativo de músicas como o da empresa "Sua Música", busquei implementar a arquitetura "Clean" no projeto seguindo as recomendações e padrões da comunidade Flutter e do Uncle Bob.

## Funcionalidades Criadas

- Tela de Splash
- Tela de Onboarding
- Tela de Escolha de Tema
- Tela de Login
- Tela de Sign In
- Tela de Sign Out
- Tela com as musicas adicionadas no FireStore
- Conexão com Firebase
- Conexão com FireStore
- Conexão com Firebase Authetication

## Tecnologias Utilizadas

- Flutter
- Bloc para gerenciamento de estado
- Firebase para gerenciamento de dados
- Flutter Screenutil para o aplicativo se adaptar e ser mais responsivo a uma gama maior de dispositivos

### Considerações Finais

Este projeto foi uma excelente oportunidade para aperfeiçoar meus conhecimentos de Flutter, gerenciamento de estado com Bloc e gerenciamento de dados usando o Firebase.

Agradeço por ter chegado até aqui, estou sempre em busca de novos conhecimentos e oportunidade, estou à disposição para quaisquer dúvidas ou esclarecimentos.
